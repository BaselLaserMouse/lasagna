Usage Instructions

Starting

See the installation guide for getting started. To get a feel of the viewer with some demo data start Lasagna with the -D flag.

Navigating 3D data

When the mouse is moved over one 2D view a set of cross-hairs appear. The other two axes show two orthogonal slices taken at the planes of the X and Y cross-hairs. As the cross-hairs are moved across the image, the slices shown in the other two plots update automatically. Zooming is performed with ctrl+mousewheel and panning by clicking and dragging. If you lose the sample due to zooming out too much, hit the reset (green circular arrow) button in the toolbar

You can change the intensity range of the image by moving the sliders on the histogram

Overlaying data

Once a 3D volume is loaded it becomes possible to load an overlay volume. The overlaid volume must have the same size as the volume upon which you wish to overlay it. Once an overlay is loaded, the first loaded layer turns red and the new layer is shown in green. Overlapping regions are yellow. The remove the overay, use the remove overlay button in the toolbar.

Using plugins

Plugins are started by selecting them from the plugin menu. Once started, a tick sybol appears next to the plugin's entry in the menu. Plugins can be stopped by unchecking the menu or, where appropriate, closing the plugin's window. Some plugins, such as the Allen Reference Atlas explorer are shut automatically if the user attempts to load a new data set.

Customisation

When Lasagna is first loaded is creates a .lasagna hidden directory in your home directory. This contains a plain text configuration file called lasagna_prefs.yml. The contents of this file are fairly self-explanatory and you may edit it. Parameters from the file can be loaded as needed during run time. So in some cases your changes will take effect right away.
