Installing

All platforms

Lasagna requires Python 2.7 and the following Python modules:

  • tifffile [for importing TIFF files]
  • vtk [for importing MHD files]
  • numpy
  • pyqtgraph (v9.10 and above)
  • yaml
  • PyQt4
  • tempfile
  • urllib

Linux & Mac

  • Download the archive or pull the Master branch from Github into a directory of you choice.
  • Install the required Python modules with your package manager or Pip. This is easy on Linux but can be a pain on the Mac. PyQt4 in particular is a challenge.
  • Run lasagna.py from within the source directory or symlink it to a location within your path. e.g.
    ln -s /home/user/code/lasagna/lasagna.py /usr/local/bin/lasagna

Windows

  • Download the archive or pull the Master branch from Github into a directory of you choice.
  • Install Python 2.7 and the required Python modules. In the past I've had good luck with modules from here.
  • Double-click on lasagna.py. You may make a shortcut to a location that suits you. Report back if there are problems (the Windows instructions haven't been tested).
