Installing

There is currently no installer. Lasagna is still aimed at more advanced users or Python developers. Lasagna uses Python 2.7 and PyQt4. To run Lasagna, download the repository and make sure you have the following Python modules installed:

  • tifffile [for importing TIFF files]
  • vtk [for importing MHD files]
  • numpy
  • pyqtgraph 9.10
  • yaml
  • PyQt4
  • tempfile
  • urllib
